# Employee Management System

## Overview

Employee Management System built using Spring Boot, Spring Security, JWT, Angular and MySQL.

## Tech Stack

* Java 17
* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA
* MySQL
* Angular
* Maven
* Git & GitHub

## Features

* User Registration
* User Login
* JWT Authentication
* Role Based Authorization
* Employee CRUD Operations
* Search Employees
* Pagination
* Google OAuth2 Login
* Profile Management
